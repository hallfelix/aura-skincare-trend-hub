## Overview

Aura is a skincare recommendation website made for people who want a more curated way to discover beauty products. Rather than sorting through broad lists, visitors can choose a skin type and receive suggestions that are better aligned with their needs.

The site uses a simple web interface and HTML-based layout, so it stays easy to navigate and fast to understand. Aura also aims to keep recommendations in step with evolving beauty trends, allowing the product set to shift as preferences change.
